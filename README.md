# Installation

`npm i easyBoxShadowBitche --save`

Then...

```
import { easyBoxShadowBitche } from "easyboxshadowbitche";

easyBoxShadowBitche({
    shadow_type: 'soft',
    padding: true
})
```

## Options

easyBoxShadowBitche supports 2 optional options :

* *shadow_type* - _soft | other_
* *padding* - _boolean_
